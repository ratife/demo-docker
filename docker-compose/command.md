# Command docker compose

## créer et starter les services

```cmd
docker-compose up -d
```

## créer et starter les services et forcer le build
```cmd
docker-compose up -d --build
```

##  starter les services
```cmd
docker-compose start -d 
```
## stopper les service
```cmd
docker-compose stop
```
## stopper et supprimer les services
```cmd
docker-compose down
```
